## Quick Start

1. Clone this repo and `cd` into `sopora-frontend`
2. `npm install`
3. `npm start`
4. Visit [http://localhost:3000](http://localhost:3000)
5. Update `.env` with any variables if needed

## Built with
- React 18+
- Axios
- React Router v6
- Context API (for auth)
- CSS Modules/global
